
# CafeTalks

a community driven social platform


## Features

- Light/dark mode toggle
- Live previews
- Cross platform
- infinite scroll


## Installation

https://reactnative.dev/docs/environment-setup

use
- yarn in project
- for android yarn android
- for ios yarn ios
